How it works

Three steps to frictionless reading

From install to your first sidebar reply in under a minute.

1
Install the extension

Add it from Chrome Web Store or Firefox Add-ons. No account, no permissions beyond bsky.app.

2
Browse Bluesky normally

Go to bsky.app and scroll your feed as usual. The extension runs silently in the background.

3
Hover any post

A sidebar slides in with all replies, threaded and live — no click, no navigation, no lost scroll position.

Features

Everything you need to read faster

🖱️
Hover to Preview

Replies appear on hover with a 400ms delay — fast enough to feel instant, slow enough to avoid noise.

🧵
Threaded Replies

Full reply tree with nested sub-threads. Expand any reply to see its own replies inline.

⚡
Smart Caching

Replies are fetched once and cached. Re-hover the same post and it loads instantly.

🔒
100% Private

Calls the public Bluesky API directly. Nothing is sent to any third-party server.

❤️
Like Counts

See like and reply counts on each reply at a glance without opening the post.

💸
Free Forever

No subscription, no premium tier, no usage cap. Open and free.

Privacy

Your browsing stays yours.

The extension only runs on bsky.app and only talks to Bluesky's own public API.

No Third-Party Servers

Replies are fetched directly from public.api.bsky.app — the same API Bluesky uses. Nothing passes through any intermediate server.

No Data Collection

The extension collects zero usage data, analytics, or personal information. It has no backend whatsoever.

Minimal Permissions

Only requests access to bsky.app. It cannot read other tabs, your history, or any other websites.
